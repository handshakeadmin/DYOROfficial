Perform an accessibility audit for WCAG 2.1 AA compliance.

1. **Semantic HTML**:
   - Check heading hierarchy
   - Verify landmark regions (main, nav, footer)
   - Review list and table structures

2. **Keyboard Navigation**:
   - Test tab order
   - Check focus indicators
   - Verify skip links

3. **Screen Reader Support**:
   - Review ARIA labels and roles
   - Check alt text for images
   - Verify form labels

4. **Color & Contrast**:
   - Check text contrast ratios (≥4.5:1)
   - Review color-only indicators
   - Test with color blindness simulators

5. **Interactive Elements**:
   - Verify button and link accessibility
   - Check modal focus management
   - Review form error handling

6. **Mobile Accessibility**:
   - Touch target sizes (≥44x44px)
   - Pinch-to-zoom not disabled
   - Orientation support

Provide specific fixes with code examples for each issue found.
